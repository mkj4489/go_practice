# go_practice

https://golang.org/

- バージョンの確認
```

go version
```
- ビルド
```
go build hello.go
出力する実行ファイルのファイル名を指定して、ビルド
go build -o hello hello.go
実行
./hello
```
- ビルド、実行
```
go run hello.go
```
- goプログラミングにオートコンプリート機能を実現するためのコマンドラインツール
```
go get github.com/nsf/gocode
```
